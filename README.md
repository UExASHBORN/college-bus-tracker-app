# College Bus Live Tracker

A complete system for real-time tracking of college buses using an ESP32 microcontroller, a NEO-6M GPS module, Firebase Realtime Database, and a web-based frontend.

## Project Structure
```text
College-Bus-Live-Tracker
├── docs/             # Documentation (Architecture, Setup)
├── firebase/         # Firebase configuration files
├── firmware/         # ESP32 Arduino sketch
├── frontend/         # Web application (HTML, CSS, JS)
├── .gitignore
└── README.md
```

## Getting Started

1. **Hardware/Firmware:** Read [docs/setup.md](docs/setup.md) for wiring and flashing instructions.
2. **Architecture:** Read [docs/architecture.md](docs/architecture.md) for a system overview.
3. **Frontend:** Read [frontend/README.md](frontend/README.md) for web app development instructions.
